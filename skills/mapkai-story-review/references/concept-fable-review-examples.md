# Concept Fable Review Examples

This file provides review examples for MapKAI concept fables.

Use these examples when a concept fable is structurally present but the decision is difficult. Compare the submitted story against these samples before choosing `PASS`, `PASS / LOCAL EDIT`, `LOCAL EDIT`, or `FAIL`.

## Contents

1. Weak But Understandable
2. Structurally Correct But Rushed
3. Beautiful But Concept-Inaccurate
4. Excellent Concept Fable

## Review Example 1: Weak But Understandable

### Input Story

```text
知识寓言
教育学

慢慢拿走的手

寓言故事
一个学生不会做木工，师傅先扶着他的手锯木头。后来师傅慢慢松手，学生就会了。最后他们发现，这就是支架式教学。

揭示的概念
支架式教学

概念解释
支架式教学是先提供帮助，再逐渐撤掉帮助。

隐喻对应
* 师傅的手：教师支持
* 锯木头：学习任务
* 松手：撤去支架

类比边界
这只是一个比喻，现实教学更复杂。

反思问题
你会如何使用支架式教学？
```

### Review Decision

LOCAL EDIT

### Reason

The story is understandable but too thin. It has only one action, one quick turn, and an immediate concept reveal. The metaphor is basically correct, but the reader does not experience temporary support becoming gradually unnecessary. The analogy boundary and reflection question are generic.

### Required Fix

Expand narrative development before concept reveal. Add at least two task moments where support changes form: full guidance, partial cue, then a question or tool that helps the learner act independently. Replace the boundary with a specific caution: scaffolding is not doing the work for the learner, and removal must match readiness.

## Review Example 2: Structurally Correct But Rushed

### Input Story

```text
知识寓言
组织管理学

两座码头

寓言故事
一家港口公司有旧码头和新码头。旧码头修熟悉的船，新码头试验新航线。经理一开始用同一张评分表考核两边，旧码头表现很好，新码头却因为害怕失败不敢试。后来经理给旧码头保留准时率和成本指标，给新码头设置实验次数和学习记录。港口终于能同时稳定赚钱和寻找未来。这就是组织双元性。

揭示的概念
组织双元性

概念解释
组织双元性指组织同时利用已有能力和探索新机会。

隐喻对应
* 旧码头：利用式活动
* 新码头：探索式活动
* 两张评分表：不同管理逻辑

类比边界
双元性不是把组织简单分成两半，还需要连接机制。

反思问题
你的组织是否用同一张评分表要求两种不同工作？
```

### Review Decision

PASS / LOCAL EDIT

### Reason

The structure is correct and the metaphor fits the concept, but the story is rushed. It compresses the evidence pressure into one sentence and reaches the concept name before the reader has spent enough time inside the two incompatible rhythms.

### Required Fix

Expand narrative development before concept reveal. Add one failed customer order, one meeting where both sides speak reasonably but miss each other, and one transfer mechanism that links exploration back to exploitation. Keep the existing metaphor.

## Review Example 3: Beautiful But Concept-Inaccurate

### Input Story

```text
知识寓言
心理学

两条河之间

寓言故事
山谷里有两条河，一条流向旧村，一条流向新城。摆渡人每天看着两边的人互相误解，于是修了一座桥。桥修好后，旧村的人和新城的人开始交换食物、消息和节日。摆渡人明白，真正困住他们的不是河水，而是缺少连接。后来，这被称为认知失调。

揭示的概念
认知失调

概念解释
认知失调指人的想法和行为发生冲突时产生的不舒服。

隐喻对应
* 两条河：冲突的认知
* 摆渡人：调节机制
* 桥：新的理解

类比边界
现实心理比桥更复杂。

反思问题
你会为自己的两条河修桥吗？
```

### Review Decision

FAIL

### Reason

The story is attractive, but the concept mapping is wrong. It is mainly about disconnected groups becoming connected. It does not stage cognitive dissonance: no conflict between belief, action, and self-image; no discomfort; no rationalization; no attempt to reduce inconsistency.

### Required Fix

Return to the concept core tension before rewriting. Use conflicting ledgers, public values vs private action, or repeated self-justification. Do not rescue the story by changing only the concept explanation.

## Review Example 4: Excellent Concept Fable

### Input Story

```text
知识寓言
传播学

杯子下面的纸条

寓言故事
社区电台每月有一次圆桌会，旧木桌上放着投票纸、咖啡杯和一只录音笔。这次议题是晚间节目要不要改版。主持人相信流程很公平，每个人都能发言，最后再看多数意见。这个办法平时管用。谁想加音乐，谁想保留访谈，都可以把理由说出来。

第一轮很快有了方向。支持改版的人坐在靠门的一侧，其中两位是常来帮忙的志愿者，说话熟，笑声也响。他们说旧节目太慢，年轻听众不爱听。有人点头，主持人在白板上写下「需要更新」。反对改版的几个人没有马上说话。一个退休教师摊开纸条，上面写着「老人晚上靠这个节目听本地新闻」，可听见旁边有人说「不能总照顾老习惯」时，她把纸条往杯子下面压了压。

主持人以为沉默就是没有强烈反对，于是进入第二轮。支持者开始引用第一轮的气氛，说大家其实都知道该改。坐在窗边的年轻剪辑师原本也担心改版太快会丢掉老听众，但他看见反对者没人开口，就把话换成了「也许可以先试」。这句话被白板记成「倾向支持试行」。

第三轮更微妙。反对者并不是被说服了，而是在计算说出口的代价。退休教师担心自己显得守旧，另一位商店老板担心以后还要跟志愿者合作，坐在后排的学生则怕自己听起来不懂媒体。越没有人反对，支持改版就越像共识；越像共识，反对者越难把第一句话说出口。

会后，主持人看着白板，支持项写满半边，反对项只有一句「照顾原听众」。录音笔里也是这样，声音多的一边像真的更大。直到清理桌子时，他看见杯子下面压着几张没交出的纸条，才意识到会议记录捕捉到的不是全部意见，而是那些敢进入空气的意见。后来，传播学把这种害怕孤立导致沉默、沉默又让多数显得更强的循环叫作沉默的螺旋。

揭示的概念
沉默的螺旋

概念解释
沉默的螺旋指人们感到自己的意见可能处于少数、会遭遇孤立时，倾向于减少表达；这种沉默又会让多数意见看起来更强，从而进一步压低少数声音。

隐喻对应
* 白板和录音笔：公共可见意见
* 杯子下面的纸条：未表达的私人意见
* 三轮发言：意见气候如何逐步改变表达意愿

类比边界
沉默不一定都来自害怕孤立，真实场景还会受到权力关系、议题敏感度、表达渠道和群体信任影响。

反思问题
当你判断「大家都同意」时，你看到的是白板上的声音，还是也会去找杯子下面的纸条？
```

### Review Decision

PASS

### Reason

The fable has a concrete scene, a reasonable old procedure, repeated friction, visible opinion climate, late concept reveal, clear mapping, and meaningful boundary. The reader can sense the concept before it is named.

### What This Sample Teaches

- Excellent concept fables do not need dramatic villains.
- Repeated rounds can make a social feedback loop visible.
- The title object returns at the end and carries the concept.
- The reflection question reuses the story object instead of asking a broad application question.
