# MapKAI Concept Fable Failure Patterns

Use this file when a concept fable feels correct but weak, rushed, generic, or not engaging.

## Contents

1. Reveal Too Early
2. Short Example + Label
3. Only One Moment Of Tension
4. Metaphor Too Generic
5. Concept Explanation Rescues The Story
6. Story Is Attractive But Concept Is Inaccurate
7. Empty Analogy Boundary
8. Reflection Question Too Broad

## Failure Pattern 1: Reveal Too Early

### Symptoms

- The title gives away the concept.
- The first paragraph already states the abstract structure.
- The reader has no discovery experience.

### Repair

- Make the title more concrete or mysterious.
- Delay concept-like terms.
- Let the old way of acting run for longer before the first friction.

### Example

Weak:

```text
标题：沉默的螺旋会议

会议一开始，大家都害怕表达少数意见，所以沉默越来越多。主持人发现这就是沉默的螺旋。
```

Better:

```text
标题：杯子下面的纸条

圆桌会开始时，主持人只看见白板上的发言越来越偏向一边。直到散会后，他才发现几张写满反对理由的纸条一直压在杯子下面。
```

## Failure Pattern 2: Short Example + Label

### Symptoms

- One scene, one problem, concept name.
- Feels like a textbook example.
- No repeated friction.

### Repair

- Add a second scene or repeated instance.
- Let the old method work once, then fail under a slightly different condition.
- Show the pattern before naming it.

### Example

Weak:

```text
老师帮学生做题，然后慢慢少帮一点。学生最后会了。这叫支架式教学。
```

Better:

```text
第一次，老师留下完整解题框架，学生能把空格补完。第二次，老师只留下关键条件，学生卡在选择公式。第三次，老师不写公式，只问他图上哪一段长度没有被使用。帮助没有突然消失，而是一层层退后。
```

## Failure Pattern 3: Only One Moment Of Tension

### Symptoms

- The conflict appears and resolves too quickly.
- The concept is understood, but not felt.
- The story jumps from first problem to concept reveal.

### Repair

- Add repeated friction.
- Let the character adjust once before the deeper pattern appears.
- Make the second friction reveal a pattern, not merely repeat the first event.

### Example

Weak:

```text
新码头试验失败，经理于是知道旧指标不适合创新。这叫组织双元性。
```

Better:

```text
第一次失败只是拖慢交付，经理以为是试验团队不熟练。第二次，旧客户提出低排放改造，旧码头按稳定流程接不住，新码头又因成本指标不敢试。经理才看见，问题不是某次失败，而是一套指标把两种工作压成了同一种节奏。
```

## Failure Pattern 4: Metaphor Too Generic

### Symptoms

- The same metaphor could explain many concepts.
- "Bridge," "map," "mirror," "river," or "maze" appears without concept-specific work.
- The story says "connection," "growth," "change," or "balance" but not the exact structure of the concept.

### Repair

- Make the key object correspond to the exact concept.
- If the concept is cognitive dissonance, the metaphor must involve conflicting accounts, values, actions, or self-justification.
- If the concept is scaffolding, the metaphor must involve temporary support and gradual removal.
- If the concept is spiral of silence, the metaphor must show expression changing perceived opinion climate, then perceived climate changing later expression.

### Example

Weak:

```text
一座桥连接两岸，人们终于理解彼此。这可以表示社会资本、结构洞、跨部门协作、传播、心理安全，几乎什么都能表示。
```

Better:

```text
协调员不只是搭桥。她知道东楼表格里哪一栏会被西楼误读，也知道西楼的护理记录怎样提前帮助东楼判断床位。她的位置有价值，是因为两个密集圈子之间本来缺少直接连接。这更接近结构洞。
```

## Failure Pattern 5: Concept Explanation Rescues The Story

### Symptoms

- The story is vague, but the explanation is correct.
- Without the explanation, the reader would not know what changed.
- The final concept section supplies the real meaning the body failed to stage.

### Repair

- Rewrite the story body.
- The story must stage the concept before the explanation names it.
- Move key concept forces into objects, repeated actions, or decisions inside the fable.

### Example

Weak:

```text
工坊里大家一开始做得不好，后来换了方法，合作变好了。

概念解释：组织双元性指组织同时利用现有能力和探索新机会。
```

Better:

```text
南码头按准点钟声修旧船，北码头试验远洋新燃料。用同一张准时率和成本表考核两边后，南码头越来越稳，北码头越来越不敢试。故事主体已经让利用和探索的冲突出现，概念解释只负责命名。
```

## Failure Pattern 6: Story Is Attractive But Concept Is Inaccurate

### Symptoms

- The story is engaging but maps poorly to the concept.
- The conflict is dramatic but not concept-driven.
- The metaphor mapping must stretch to fit the concept.

### Repair

- Identify the concept's true core tension.
- Remove unrelated drama.
- Rebuild around the actual theoretical structure.

### Example

Weak:

```text
为了表现认知失调，故事写一个人在两份工作之间犹豫，最后选择更赚钱的一份。故事有选择压力，但没有信念、行动和自我形象的冲突，也没有自我合理化。
```

Better:

```text
馆员相信自己在保存城市记忆，却签字卖掉一些并非真正重复的旧信。随后他不断改写「保存」的含义，让行动和自我形象不再互相刺痛。冲突来自认知失调本身。
```

## Failure Pattern 7: Empty Analogy Boundary

### Symptoms

- Boundary section says only "this is just a metaphor."
- It does not explain what the metaphor cannot show.
- It fails to prevent overclaiming.

### Repair

- State what the concept does not mean.
- State where the analogy may mislead.
- Mention the condition, limit, or missing factor.

### Example

Weak:

```text
类比边界：这只是一个比喻，现实更复杂。
```

Better:

```text
类比边界：心理安全不等于没有责任，也不等于所有错误都可以被原谅。它强调的是在追责之前，让风险和疑问能更早进入团队。
```

## Failure Pattern 8: Reflection Question Too Broad

### Symptoms

- "What do you think?"
- "How can you apply this?"
- The question does not return to the story object.
- The question loses the concept's core dilemma.

### Repair

- Use the story's central object, tension, or choice.
- Ask a question that preserves the concept's core dilemma.
- Make the reader re-enter the metaphor with their own situation.

### Example

Weak:

```text
你会如何在生活中使用这个概念？
```

Better:

```text
当你的团队只看见白板上说出口的意见时，你会不会去找一找那些还压在杯子下面的纸条？
```
